# ДЗ 10 — PoC для CVE-2019-19781

Репозиторий для домашнего задания №10 по курсу Python для кибербезопасности.  
Задача: выбрать известную уязвимость и реализовать простой proof-of-concept-скрипт, который демонстрирует идею атаки (без эксплуатации в бою).

В качестве примера выбрана уязвимость **CVE-2019-19781** в продуктах **Citrix ADC / Citrix Gateway**.

---

## Описание уязвимости

CVE-2019-19781 — это уязвимость типа **directory traversal / path traversal** во веб-компонентах Citrix ADC / Citrix Gateway.

Проблема в том, что сервер некорректно фильтрует последовательности вида `../` в URL.  
Из-за этого удалённый неаутентифицированный пользователь может сформировать специальный запрос и получить доступ к файлам за пределами корректной веб-директории. В некоторых конфигурациях это приводит к чтению конфиденциальных файлов и дальнейшему выполнению произвольного кода.

Классический пример проверки — запрос к конфигурационному файлу `smb.conf` по пути:

```text
/ vpn / ../ vpns / cfg / smb.conf
