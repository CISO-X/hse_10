# ДЗ 10 — PoC для CVE-2019-19781

Репозиторий для домашнего задания №10 по курсу «Python для кибербезопасности».

В качестве примера выбрана уязвимость **CVE-2019-19781** в **Citrix ADC / Citrix Gateway**.  
Это уязвимость типа *directory traversal*: сервер некорректно обрабатывает последовательности `../` в URL, что позволяет получать доступ к файлам за пределами веб-директории.

## Что делает скрипт

Файл `poc_cve_2019_19781.py`:

1. Принимает базовый URL сервера (например, `https://example.com`).
2. Собирает тестовый путь `/vpn/../vpns/cfg/smb.conf`.
3. Отправляет HTTP-запрос `GET` на этот URL.
4. Анализирует ответ и выводит:
   - найден ли в ответе фрагмент, похожий на конфиг `smb.conf`;
   - сообщение о том, есть ли признаки CVE-2019-19781.

Скрипт ничего не эксплуатирует, а только проверяет поведение сервера.

# или

# и затем ввести URL вручную
