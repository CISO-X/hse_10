1. Описание уязвимости (текст для README / отчёта)

CVE-2019-19781 — Directory Traversal в Citrix ADC / Citrix Gateway

Продукт: Citrix Application Delivery Controller (ADC) и Citrix Gateway

Тип уязвимости: Path Traversal / Directory Traversal (CWE-22)

Затронутые версии (основные): 10.5, 11.1, 12.0, 12.1, 13.0 до установки патчей.

Суть проблемы:
Веб-компонент, отвечающий за VPN-функциональность, некорректно фильтрует последовательности вида ../ в URL.
Это позволяет удалённому неаутентифицированному атакующему сформировать запрос с переходом «выше» корневого каталога и читать файлы за пределами веб-директории. В ряде сценариев это приводит к выполнению произвольного кода на устройстве.

Примером злоупотребления этой уязвимостью является запрос к файлу конфигурации smb.conf по пути:

/vpn/../vpns/cfg/smb.conf

Если сервер возвращает содержимое с типовой строкой [global], это индиректно подтверждает наличие уязвимости. Именно такой подход используется в сканере Metasploit для данной CVE.

Важно: Эксплуатация уязвимости на реальных чужих системах без явного разрешения — незаконна. PoC ниже предназначен только для проверки собственных систем в учебных целях, как требует формулировка ДЗ.
